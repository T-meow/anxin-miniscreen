## 二、两篇必发帖子的模板

### 帖子A：报名帖（创意提案）

**发布位置**：TRAE 社区 → 【大赛报名专区】
**标签**：`社会服务` + `社会公益`
**标题**：`【社会服务赛道】安心小屏 - 旧手机改造的老人陪伴与远程照护助手`

**正文结构**：
1. 创意名称 + 创意介绍
2. 目标用户及痛点
3. 价值与意义
4. 技术方案概述

> 注意：报名帖必须先审核通过，初赛 Demo 帖才有效。

### 帖子B：初赛 Demo 帖（作品帖）

**发布位置**：TRAE 社区 → 【大赛初赛专区】
**标签**：`社会服务` + `社会公益`
**标题**：`【社会服务赛道】安心小屏 Demo - 旧手机改造老人陪伴助手（附在线体验）`

**正文结构**（按官方模板）：

**1. Demo 简介**
- 是什么：PWA 网页应用，将旧安卓手机改造成老人桌面陪伴屏
- 面向谁：独居老人、留守老人、异地照护的子女
- 主要功能：提醒系统、AI问助手、娱乐陪伴、视频通话、隐私保护
- 附产品截图

**2. Demo 创作思路**
- 灵感来源：观察父母不会用智能手机、子女不在身边的困境
- 想解决的问题：老人不会用App、子女远程关心困难、提醒吃药靠记忆
- 为什么做这个方向：低成本、零安装、适老化

**3. Demo 体验地址**
- 部署后的 Vercel 链接
- 或打包 ZIP 上传 HTML 文件

**4. TRAE 实践过程**
- 开发关键步骤截图（≥3张）
- 关键任务对话的 Session ID（≥3个）
- 附报名帖链接
