# 安心小屏 - 比赛演示指南

## 项目概述

安心小屏是一款面向独居老人、留守老人和异地照护家庭的低成本陪伴产品。核心思路是将闲置旧安卓手机改造成长期摆放在老人家中的桌面陪伴小屏，通过大字体、大按钮、语音操作降低使用门槛。

## 技术栈

- **前端**: PWA (HTML/CSS/JS)
- **语音**: Web Speech API (本地 ASR/TTS)
- **视频通话**: WebRTC (演示模式)
- **AI 问答**: 本地响应 + 模拟 AI
- **数据同步**: localStorage + BroadcastChannel

## 演示流程

### 场景：子女远程关心独居母亲

#### 步骤 1：子女端添加提醒
1. 切换到"子女端"
2. 点击"晚 8 点吃药"快捷按钮，自动填充表单
3. 点击"保存提醒"
4. 观察右侧提醒列表显示"待完成"

#### 步骤 2：老人端查看提醒
1. 切换回"老人端"
2. 观察"下一项提醒"面板显示刚刚添加的提醒
3. 点击"今日提醒"大按钮，查看提醒详情

#### 步骤 3：提醒触发（模拟）
1. 等待约 2 分钟（演示数据设置的提醒时间）
2. 或手动修改系统时间到提醒时间
3. 观察老人端自动弹出大字提醒弹窗
4. 弹窗包含：提醒标题、说明、"我吃过了"按钮

#### 步骤 4：老人确认提醒
1. 点击"我吃过了"按钮
2. 听到语音播报"好的，已记录"
3. 弹窗关闭

#### 步骤 5：子女端查看确认状态
1. 切换到"子女端"
2. 观察提醒列表状态变为"已确认"
3. 设备状态显示"刚刚有互动"

#### 步骤 6：老人使用 AI 助手
1. 切换回"老人端"
2. 点击"问助手"大按钮
3. 输入"明天会下雨吗"
4. 观察 AI 回复天气信息
5. 听到语音播报回复内容

#### 步骤 7：老人听故事
1. 点击"听一会儿"大按钮
2. 选择"睡前故事：小猫钓鱼"
3. 观察播放器界面
4. 点击播放/暂停按钮

#### 步骤 8：子女发起视频通话
1. 切换到"子女端"
2. 点击"发起视频"按钮
3. 观察呼叫界面"正在呼叫陈阿姨"
4. 等待 2 秒后自动切换到老人端来电界面

#### 步骤 9：老人接听视频
1. 观察老人端显示"儿子来电"
2. 注意隐私状态条变红闪烁
3. 点击接听按钮
4. 进入通话中界面，显示计时器

#### 步骤 10：结束通话
1. 点击挂断按钮
2. 观察隐私状态条恢复绿色
3. 回到老人端首页

## 核心亮点

### 1. 适老化设计
- 大字体、大按钮，减少误触
- 高对比度配色，视力友好
- 语音播报，无需阅读

### 2. 隐私保护
- 摄像头/麦克风使用状态始终可见
- 通话时红色闪烁提示
- 本地语音识别，数据不上传

### 3. 低成本方案
- 利用闲置旧手机，零硬件成本
- PWA 无需安装，浏览器打开即用
- 纯前端实现，无需后端服务器

### 4. 亲情连接
- 子女远程设置提醒
- 老人确认状态实时同步
- 一键视频通话

## 启动方式

### 前端
```bash
cd src
npx serve . -p 8080
# 访问 http://localhost:8080
```

### 服务端（可选）
```bash
cd server
npm install
npm start
```

## 注意事项

1. **演示模式**：当前为演示版本，AI 响应为模拟数据，视频通话为界面流程演示
2. **浏览器兼容**：建议使用 Chrome/Edge 最新版本
3. **语音功能**：需要浏览器支持 Web Speech API
4. **数据存储**：使用 localStorage，刷新页面数据不丢失
5. **重置数据**：在老人端"简单设置"中点击"重置演示数据"
